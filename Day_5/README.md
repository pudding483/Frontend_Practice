1. 狀態選取器
2. Transition 漸進/漸出
   - 練習使用 transition 搭配 :hover，實作以下效果：
     - 滑鼠移上卡片時：
       - 卡片放大（scale）
       - 陰影變深
       - 背景顏色改變
       - 動畫要有流暢的過渡效果
3. Animation 動畫
   - 使用 animation + position 製作一個彩色方塊，具備以下效果：
     - 從 左邊移動到右邊接著再從右邊再回到左邊
     - 無限次重複
   - 在移動過程中：
     - 顏色逐漸改變（如從紅色 → 黃色 → 藍色）
     - 大小逐漸放大（再縮回）
4. RWD 響應式網頁設計
   - 做一個導覽列（nav）(with media query)
     - 桌機顯示橫向排列的選單項目
     - 手機螢幕下變成直向排版
     - 加上顏色變化，讓不同螢幕狀態明顯區分
